因為使用方法很類似我們來看幾個實際使用範例。



```python
from sklearn.datasets import make_classification

X, y = make_classification(n_samples=1000, 
                   n_features=7,
                   n_informative=4,
                   n_classes=3)

X.shape, y.shape
```


```python
from sklearn.datasets import make_regression

X, y = make_regression(n_samples=1000, 
                   n_features=7,
                   n_targets=2,
                   random_state=87)

X.shape, y.shape
```


