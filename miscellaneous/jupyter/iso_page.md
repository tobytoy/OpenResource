# 搶 $15$
我們有數字 $1$ 到 $9$，兩個人輪流拿不可重複拿取，誰拿到數字任三個和為 $15$的人贏。

## Example
假設兩個人叫 $A,B$ ， $A$ 搶了三個數字 $1, 5, 9$ ， $B$ 搶了三個數字 $2, 3, 4$ ，
則 $A$ 勝利。

請問這個遊戲有必勝法嗎?



# 解

|  |  |  |
| :-----: | :----: | :----: |
| 4 | 9 | 2 |
| 3 | 5 | 7 |
| 8 | 1 | 6 |
